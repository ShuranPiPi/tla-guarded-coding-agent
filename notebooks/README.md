# Notebooks

Maintained demos:

- `01_formal_spec_guarded_agent_demo.ipynb`: runs the spec-first agent on a sample task with a selectable spec mode; `example` uses TLC and `specification` uses TLAPS.
- `02_provider_fallback_demo.ipynb`: inspects OpenAI/Gemini provider selection.
- `03_spec_mode_comparison.ipynb`: compares deterministic TLC and TLAPS checker paths without requiring an LLM key.

Historical draft notebooks and the old README live in `archive/`. They are
preserved for reference only and are not expected to match the current package
API or the current spec-first workflow.
